# Run project on localhost

Run  **create** and **migrate** commands:

    python manage.py makemigrations
    python manage.py migrate
    
Import **file.csv**:

    python manage.py importcsv path_file.csv
    
Run server:
    
    python manage.py runserver 

## Technologies:
- Python 3.5, Django 1.11
- jQuery, highcharts
